## 1. Server
./ircserv <port> <password>

## 2. Client

irssi -c <hostname> -p <port> -w <password> -n <nickname>

## 3. Commands
/connect <hostname> <port> <password> <nickname>
/join <channel>
/part <channel>
/msg <nickname> <message>
/topic <channel> <topic>
/mode <channel> <mode>
/kick <channel> <nickname>
/invite <nickname> <channel>
/quit
